# Calculator App

A calculator app with basic math funktions as buttons and a display windows to display the result. Widgets used in this project.
- Scaffold
- AppBar
- Padding
- Icon
- Color
- SizedBox
- SafeArea
- Column
- Align
- Row
- TextWidgets
